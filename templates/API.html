Инструкция по использованию API:



API обращается к главному окну сайта, следовательно, если вы хотите загрузить список с лекциями, то вам необходимо составить запрос следующего типа:

1. http://127.0.0.1:5050/main (если вам необходим список со всеми лекциями);
2. http://127.0.0.1:5050/main/название урока (если вам необходима конкрентная лекция)
